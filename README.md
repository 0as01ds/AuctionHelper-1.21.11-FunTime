# AuctionHelper-1.21.11-FunTime
Мод, позволяющий видеть цену за 1 товар на аукционе сервера FunTime. 
https://github.com/SuckS1de - Взял исходники у него. 
Добавил: Категории (чтобы показ цен был в категориях товаров), скомпилировал, оба файла перекиуть в Mods.
НЕ ЯВЛЯЕТСЯ ПОЛНЫМ ПРОЕКТОМ - КОПИРУЕТЕ 1 ФАЙЛ И ВСТАВЛЯЕТЕ В ОФФИЦИАЛЬНУЮ ВЕРСИЮ "AuctionHelper-Minecraft-1.21.11--main/src/main/java/wtf/auction/helper/mixin/ ". Заменяете файл  находящийся в самом конце на мой.

![Пример работы](IMG_20260514_182717.jpg)

Для работы требуется:
*   **Fabric API** для версии 1.21.11
    *   [Скачать Fabric API (1.21.11)](https://minecraft-inside.ru/download/489259/)
*   **Java 21** или новее
    *   [Скачать Java 21+](https://www.oracle.com/java/technologies/javase/jdk21-archive-downloads.html)

Установка:
1.  Скачайте Fabric API по ссылке выше.
2.  Скачайте Java 21+ по ссылке выше.
3.  Скачайте архив из релиза, распакуйте.
4.  Поместите файлы `.jar` (в том чисте и Fabric API) в папку `mods`.
5.  Готово!
