# AuctionHelper-1.21.11-FunTime
Мод, позволяющий видеть цену за 1 товар на аукционе сервера FunTime. 
Нужен Fabric api 1.21.11 0.100+; Fabric 1.21.11; gradle; Java 21+.  
https://github.com/SuckS1de - Взял исходники у него. 
Добавил: Категории (чтобы показ цен был в категориях товаров), скомпилировал, оба файла перекиуть в Mods.
